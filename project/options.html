RaBiTool

Automacao web para levar exportacoes XLSX do Reclame Aqui para a planilha mae no Excel.

Fluxo

Origem:
Reclame Aqui, via navegacao e download XLSX
Destino:
Excel Web, via interacao de interface
Metodo:
Automacao de abas, botoes, espera de pagina e importacao/colagem

Atalhos de Teclado

Ativar / Desativar

Versao

Version/update UI is scaffolded and hidden until configured.
